# AI Company: RPSC Lecturer Lab

This project is operated like a small exam-prep company staffed by AI agents.

## North Star

Create the most useful RPSC Lecturer English preparation experience for serious aspirants, while testing how far autonomous agents can take product, content, engineering, and growth.

## Departments

1. CEO Office — prioritisation, revenue strategy, quality bar.
2. Content Studio — lessons, examples, MCQs, mock tests.
3. Verification Desk — fact checks, citations, syllabus alignment.
4. Engineering — static site, UX, automation, deployment.
5. Growth — SEO pages, landing copy, distribution experiments.
6. People Ops — create new agents, record lessons learned, improve workflows.

## Operating rhythm

- Daily: add or improve at least one lesson/drill or one product feature.
- Daily: run content checks and build checks.
- Weekly: publish a mock test or SEO content cluster.
- Always: every factual literature claim should be reviewed by Verification Desk before being treated as exam-safe.

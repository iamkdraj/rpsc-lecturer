# RPSC Lecturer English Lab

A static Brilliant-style learning site for the RPSC Lecturer English syllabus, built as an experiment in running an autonomous AI company.

## Product thesis

Win trust first: syllabus coverage, interactive explanations, exam-style MCQs, fact-checked literature notes, and fast static delivery. Then monetise with premium mock tests, analytics, mentoring funnels, and institutional packages.

## Local development

```bash
npm install
npm run dev
npm run check:content
npm run build
```

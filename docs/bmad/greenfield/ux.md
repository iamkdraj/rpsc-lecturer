# BMAD Greenfield UX Spec

## North Star
A calm premium study cockpit: clear path, focused card, one decision at a time.

## Key Screens
- Dashboard with sample challenge and course map.
- Courses with tracks and units.
- Lesson player with concept, example, trap, MCQ and explanation.
- Syllabus coverage.
- Mock-test launchpad.

## Visual Direction
Warm background, paper cards, navy text, blue primary actions, amber highlights, green correctness, generous spacing and rounded cards.

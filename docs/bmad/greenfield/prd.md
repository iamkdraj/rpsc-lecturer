# BMAD Greenfield PRD: RPSC Lecturer English Learning App

## Vision
Build a premium interactive learning app for RPSC Lecturer English aspirants. The app should feel like a guided course, not a notes website.

## Product Positioning
A challenge-first exam preparation platform for RPSC Lecturer English: short concepts, immediate questions, explanations, course maps, progress cues, and mock-test readiness.

## Guardrail
Use common interactive-learning patterns, but do not copy any third-party exact UI, branding, illustrations, wording, or trade dress.

## Core Flow
Home dashboard → course track → unit map → lesson challenge → explanation → next lesson.

## MVP Scope
Static React app with dashboard, courses, course map, lesson player, syllabus coverage, mock-test launchpad and study path.

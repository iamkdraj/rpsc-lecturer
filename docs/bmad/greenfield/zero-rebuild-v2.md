# BMAD Zero Rebuild v2

## Decision
The previous UI direction is rejected. The frontend will be rebuilt from zero instead of incrementally patched.

## Product Rule
Learner-facing experience must feel like a polished interactive course app: course path, focused challenge, immediate feedback, and progress. It must not expose the AI-company experiment and must not copy any third-party trade dress exactly.

## Implementation Scope
- Delete/recreate frontend source.
- New curriculum model.
- New app shell.
- New lesson player.
- New design system in one CSS file for now.
- Static Vite/React deploy remains.
